Exercise: Intro and Basic Syntax

1.	Ages
Write a program that determines if a person is baby, child, teenager, adult or elder based on the given age. The boundaries are:
•	0-2 – baby
•	3-13 – child
•	14-19 – teenager
•	20-65 – adult
•	>= 66 – elder
•	All the values are inclusive.
